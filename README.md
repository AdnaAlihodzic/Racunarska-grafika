# Racunarska-grafika
Projekat rađen u sklopu predmeta Racunarska grafika (V semestar).
Projekat je realizovan korištenjem alata **Blender** i obuhvata proces 3D modeliranja, primjene materijala i tekstura, renderovanja i finalne vizualne prezentacije.

Cilj projekta je primjena teorijskih i praktičnih znanja iz oblasti računarske grafike kroz izradu kompletnog 3D vizualnog rješenja.
